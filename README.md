# InsideTheCell

## Description
This project is an educational video game that combines biology and interactive gameplay to teach players how viruses infect cells.
The player explores a cell through a tutorial area, learning about its components (such as the membrane, nucleus, and organelles) and their role in defending against viruses. Once the infection begins, the player must face a series of mini-games, each representing a different phase of the virus-cell infection process: binding, entry, uncoating, replication, assembly, and release. 
Each mini-game is designed with a unique mechanic and increasing difficulty. On the first attempt, the challenge is intentionally very difficult, often leading the player to fail. After this, an interactive quiz guided by a Large Language Model (LLM) helps the player understand the biological concepts behind the failure. 
Once the player completes the quiz, they become "vaccinated", meaning they can replay the mini-game with balanced difficulty and improved chances of success. This mechanic reflects how knowledge and immunity help in fighting infections. 
The goal of the game is to successfully complete all mini-games in vaccinated mode, mastering both the gameplay and the underlying scientific concepts. Rewards such as cosmetic items can be unlocked by achieving perfect scores.
This project aims to make learning biology engaging, intuitive, and interactive by blending gaming mechanics with educational content.
